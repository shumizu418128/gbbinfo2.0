### メモ
- 2023年のみ、ruleページと出場者データ未接続
- csvには出場者未定のシード権も記入
  - その際、出場者名は「～?/? 発表～」とする
  - 出身国が不明の場合、国コードは0とする
    - 国コード0はcountry.csvに追加済み
    - 国コード0の場合、world_mapには表示されない
